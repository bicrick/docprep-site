# DocPrep Site

Landing page for DocPrep - a tool that extracts clean text from documents to make them AI-ready.

## Overview

A single-page marketing site showcasing DocPrep's features:
- Hero section with product introduction
- How it works tutorial with animated file tree comparison
- AI-ready data use case section
- Download section for macOS and Windows

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and animations
- `script.js` - Smooth scrolling, animations, and interactions
- `assets/` - Logo and icon files

## Usage

Open `index.html` in a web browser or serve via any static file server.

## Features

- Smooth scroll navigation
- Scroll-based animations
- Responsive design
- Dark theme with polka dot background
